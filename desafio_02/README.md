# Desafio 02 — Maior sequência crescente contínua

## Enunciado

Dado um array de inteiros, retornar o tamanho da maior sequência crescente contígua (elementos consecutivos no array em ordem estritamente crescente).

## Assinatura

```python
def longest_continuous_increasing(nums: list[int]) -> int
```

## Complexidade alvo

- **Tempo:** O(n)
- **Memória:** O(1)

## Principais casos de borda

- Array vazio
- Array com um elemento
- Array totalmente crescente
- Array totalmente decrescente
- Elementos iguais consecutivos
- Sequência crescente no final do array
